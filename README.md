# Numerical Stability of Recurrent Reinforcement Learning

This repository investigates the numerical stability of Recurrent Reinforcement Learning (RRL) using LSTMs within Partially Observable Markov Decision Processes (POMDPs). Specifically, it evaluates the empirical relationships between the spectral radius of the hidden-to-hidden Jacobian, hidden state drift, gradient norms, and overall policy degradation in PPO.
